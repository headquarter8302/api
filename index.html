API Endpoint

Directory

  • about.json - general data
  • quotes.json - misc/, quotes I've gathered
  • status.json - status of the endpoint, should return "OK"
  • music.json - information/, information regarding my music
  • schema.json - meta/schema/, the schema

Schema

  1. meta - object, containing metadata regarding the endpoint and the response.
    1. type - string, First Letter Capitalized, containing the type of data that's given, can be one of:
      • String
      • StringURL, a URL in the form of a string, should be a legal WHATWG URL according to the living standard
      • Integer
      • Array
      • Boolean
      • Object, will contain other types of data.
      • mixed, contains more than one property.
  2. data - object, the data object iself. Can contain anything.
